# HealthCare Dashboard

Pixel-perfect responsive healthcare dashboard built with React.

This project demonstrates modern frontend development, component-based architecture, responsive layout design, static data handling, and accurate UI implementation from a reference dashboard design.

## Overview

HealthCare Dashboard is a frontend showcase project focused on building a clean medical analytics dashboard UI with reusable components and production-style styling.

The project is useful for demonstrating:

- React component architecture
- dashboard layout design
- responsive UI implementation
- static mock data organization
- pixel-perfect CSS alignment
- frontend portfolio quality

## Features

- Responsive healthcare dashboard UI
- Sidebar navigation
- Header section
- Anatomy/health overview section
- Health status cards
- Calendar view
- Upcoming schedule section
- Activity feed
- Reusable appointment cards
- Static mock data structure
- Clean component separation

## Tech Stack

- React.js
- JavaScript
- CSS3
- Vite

## Project Structure

```txt
src/
  components/
  data/
  styles/
  App.jsx
  main.jsx
```

## Getting Started

```bash
git clone https://github.com/SunilKumarKV/HealthCareDashboard.git
cd HealthCareDashboard
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Portfolio Purpose

This repository is maintained as a frontend portfolio project to demonstrate UI precision, responsive design, and React component organization.
